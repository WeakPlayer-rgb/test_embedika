Команды:
    Print database - без параметров, выведет все машины данные по которым хранятся в базе. Для фильтрации машин нужно описать ту информацию, которую мы ищем в виду {"param1":"value1","param2":"value2" и т.д.}
    Add car Json - десериализует Json в объект машины и добавляет её в базу данных. Выкидывает ошибку если машину уже есть в базе.
    Delete car Id - удаляет машину с id из базы данных.
    Print statistic - выводит статистику по базе данных: количество машины в базе, время добавления первой записи, время добавления последней записи.
